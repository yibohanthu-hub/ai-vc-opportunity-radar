# 90 秒 Demo 演示脚本

目标：在约 90 秒内展示“发现—整理—筛选—查看—收藏—跟进”的完整闭环。以下为建议口播，括号内为屏幕操作。

| 时间 | 操作 | 口播重点 |
| --- | --- | --- |
| 0–10 秒 | 打开首页 | “这是面向 VC 投资经理的 AI 机会雷达。它不是自动投决工具，而是把公开信号整理成可验证、可跟进的项目和活动清单。” |
| 10–18 秒 | 指向今日新增项目和活动、收藏与待跟进 | “首页先给出今天新增的项目和活动数量，以及收藏和待跟进状态；下方的推荐入口直接带我进入高优先级线索。” |
| 18–30 秒 | 进入「创业项目」，筛选 `Agent 基础设施` | “先看新增项目。我用行业筛选把 23 条公开样本缩到 Agent 基础设施。排序不是黑箱：它由信息新颖性、来源、完整度、AI 相关性和跟进价值组成。” |
| 30–46 秒 | 打开 `Spine AI` 详情 | “详情页先给产品、客户、AI 用法和团队。融资和估值没找到可靠来源时就显示未知，不会补造。” |
| 46–57 秒 | 指向来源、Agent 理由、不确定性和验证问题 | “右侧不是‘建议投资’，而是为什么值得再读、当前不知道什么、以及下一步可以问什么。来源和公开联系入口可以直接打开，但系统不会自动联系任何人。” |
| 57–64 秒 | 收藏、标签、设为「待联系」 | “投资经理可以收藏、加标签并设定跟进状态。这些动作会积累成第二阶段可解释推荐的行为信号。” |
| 64–76 秒 | 切换「AI 活动」，筛选 `低` 再清除，打开 `World Summit AI 2026` | “活动模块不仅收录高价值会议，也识别低优先级活动。低价值不代表不好，而是它缺少项目发现或深度交流机制。打开高价值活动后可以看到主题、嘉宾质量、投资人价值、来源和关联项目。” |
| 76–88 秒 | 进入「收藏与跟进」，多选、批量设置状态 | “最后，收藏与跟进页把项目和活动集中起来，显示最近查看、下一步行动和公开来源。可以批量整理，不需要在多个表格之间跳转。” |
| 88–90 秒 | 回到首页或停在跟进页 | “V1 先解决可信信息池和工作闭环；下一阶段才接入合规数据源和可解释个性化推荐。” |

演示前准备：清空浏览器本地存储或使用新的浏览器 profile，以保证收藏和状态从预期初始值开始。演示中最好收藏 `Spine AI`、添加“Agent 基础设施”标签，并在活动页打开 `World Summit AI 2026`。
