## 使用原则

- 来源链接支持“从摘要回到原始证据”，不代表所有页面都已对每一个字段做独立事实审计。
- 从当前收录来源不能确认的融资、估值、团队背景、活动嘉宾或日程，Demo 一律显示未知 / 待核验。
- 活动信息时效性高；参与或联系前必须重新打开官方页面核验。
